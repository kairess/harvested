# Harvested

"Harvested"는 채팅 형식의 스토리텔링 게임입니다.

[플레이하기](https://kairess.github.io/harvested)

![Harvested](result.png)

## 🌟 주요 기능

- 채팅 형식의 스토리텔링
- 타이핑 효과
- 화면 전환 애니메이션
- 뒤로가기 기능
- URL 파라미터를 통한 특정 장면 접근 (예: ?screen=2&message=1)

## 🛠 기술 스택

- React
- Vite
- CSS Animations

## 🚀 시작하기

```bash
git clone https://github.com/kairess/harvested.git
cd harvested
npm install
npm run dev
```
